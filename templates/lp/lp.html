API Endpoint Testing

HTLogin can also test JSON API and GraphQL endpoints when forms are not found.

  • JSON API: POST /api/login (View JSON Login page)
  • GraphQL: POST /api/graphql (View GraphQL page)

Try: curl -X POST http://127.0.0.1:5000/api/login -H "Content-Type: application/json" -d '{"username":"admin","password":"admin"}'
